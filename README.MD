# Basic Music Player

This is a basic music player using Node.js and MongoDB to stream audio.

## Installation

To install locally you must have a MongoDB instance running, modify config vars in index.js for your connection string.

```bash
npm install -g
```

To use an already hosted version go to:

[This location](https://comp127musicplayer.herokuapp.com/)
## To run

```bash
node index.js
```
